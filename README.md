 # Agenda Telefónica en C#

## Descripción

Proyecto desarrollado para la asignatura de Estructura de Datos de la Universidad Estatal Amazónica.

El sistema permite:

* Registrar contactos.
* Mostrar contactos registrados.
* Buscar contactos por nombre.
* Eliminar contactos.

## Tecnologías utilizadas

* C#
* Programación Orientada a Objetos
* Listas (List)

## Estructuras de datos utilizadas

* Clase Contacto (registro de información).
* Lista dinámica para almacenar contactos.
